A simple and clean Image card which will show content on hover.

== Description ==

Beautiful Image Card is an amazing and light weight Image card that is very flexible and customizable.
You can add interactive Image card in no minute with BIC. It shows Title and as soon as you hover over it, the remaining content will be revealed. 

Beautiful Image Card can be used showcase team members, Authors, Leaders, Organizers, Features, Products etc.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/orange-image-card` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Open any page/post editor (Gutenburg) and search for 'Beautiful Image Card' block.


== Frequently Asked Questions ==

= Do I need Gutenburg for this? =

Yes, this plugin creates a block which you can access from the gutenburg editor.

= Is it Free? =

Beautiful Image Card is completely free.

= Will it impact my website performance? =
No, BIC is very fast and lightweight. I am not using any javascript, its just pure html/css
